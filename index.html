Data Visualisations by David John Wilson

This page gives a brief demonstration of the visualisations included in this repository. All visualisations are built over d3.js.

toptwenty.js: Interactive Top 20 List

toptwenty.js allows a user to easily create an interactive Top 20 list from lists of key-value pairs. The end-user can interact with the list to change the data source or the sample size, with each change handled with smooth animations. Hovering over a bar in the Top 20 displays the key information and highlights the bar.

A user can not only customise the data used, but also the sample size options and various appearance parameters (colors, speed of animations, size of visualisation, etc).

Below is an example of an interactive Top 20 list created with toptwenty.js. It displays the top choreographers (with respect to the number of pieces they have created) and the top ballets (with respect to the number of performances of each ballet) for the Royal Ballet at the Royal Opera House, London (between 1957 and 2012).
